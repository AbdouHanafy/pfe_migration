\section{Contexte du stage}
Le stage s'inscrit dans un besoin concret: accompagner la migration de machines virtuelles vers OpenShift Virtualization en reduisant les risques techniques et en ameliorant la qualite de preparation avant migration.

\section{Missions confiees}
Les missions principales confiees durant le stage sont:
\begin{itemize}[leftmargin=1.2cm]
  \item analyser les contraintes techniques de migration VM vers OpenShift,
  \item concevoir une architecture logicielle adaptee,
  \item developper un backend de traitement et d'orchestration,
  \item proposer une interface frontend de pilotage,
  \item valider le pipeline sur des cas reels de VMs.
\end{itemize}
